# AY-3-8912 Sound Chip (128K)

The General Instrument AY-3-8912 programmable sound generator (PSG) is present in all post-48K models (128K, +2, +2A, +2B, +3). It is controlled by two I/O ports:

| Port | Direction | Function |
|---|---|---|
| `0xFFFD` | OUT | Select register (0–14) |
| `0xFFFD` | IN | Read value of the selected register |
| `0xBFFD` | OUT | Write value to the selected register |

Usage: select a register by writing to `0xFFFD`, then read/write the register value via `0xBFFD` (read) or `0xFFFD` (write).
