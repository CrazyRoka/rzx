# AY-3-8912 Sound Chip (128K)

The General Instrument AY-3-8912 programmable sound generator (PSG) is present in all post-48K models (128K, +2, +2A, +2B, +3). It is controlled by two I/O ports:

| Port | Direction | Function |
|---|---|---|
| `0xFFFD` | OUT | Select register (0–15) |
| `0xFFFD` | IN | Read value of the selected register |
| `0xBFFD` | OUT | Write value to the selected register |

Usage: select a register by writing to `0xFFFD`, then read/write the register value via `0xBFFD` (read) or `0xFFFD` (write).

### Model Differences

On the ZX Spectrum 128 and +2, reading `0xBFFD` returns **floating bus** data (same as any unused port). On the +2A, +3, +2B, and +3B, reading `0xBFFD` returns the **same value** as reading `0xFFFD` (the currently selected register).

## Pinout (28-pin DIP)

```
            +------+
  SOUND C  1 |      | 28  D0
  PORT     2 |      | 27  D1
  Vcc      3 |      | 26  D2
  SOUND B  4 |      | 25  D3
  SOUND A  5 |      | 24  D4
  GND      6 |      | 23  D5
  PORT     7 |      | 22  D6
  PORT     8 |      | 21  D7
  PORT     9 |      | 20  BC1
  PORT    10 |      | 19  BC2
  PORT    11 |      | 18  BDIR
  PORT    12 |      | 17  A8
  PORT    13 |      | 16  RESET
  CLOCK   14 |      | 15  CLOCK
            +------+
```

| Pin(s) | Name | Function |
|---|---|---|
| 1 | SOUND C | Audio output channel C |
| 2, 7–12 | PORT | I/O port connections to the system |
| 3 | Vcc | +5 V power supply |
| 4 | SOUND B | Audio output channel B |
| 5 | SOUND A | Audio output channel A |
| 6 | GND | Ground |
| 13 | RESET | Reset (active low) |
| 14 | CLOCK | Clock input (system clock divided down) |
| 15 | CLOCK | Clock output |
| 16 | RESET | Reset |
| 17 | A8 | Address line 8 (register select) |
| 18 | BDIR | Bus direction control |
| 19 | BC2 | Bus control 2 |
| 20 | BC1 | Bus control 1 |
| 21–28 | D0–D7 | Data bus |

In the Spectrum, the three audio outputs (SOUND A, B, C) are typically tied together.

## Register Map

The AY-3-8912 has 16 registers (0–15). Register 15 (I/O port B) is not connected on the 8912 variant.

| Register | Name | Width | Description |
|---|---|---|---|
| 0 | Channel A fine pitch | 8 bits | Tone period for channel A (low byte) |
| 1 | Channel A coarse pitch | 4 bits | Tone period for channel A (high nibble) |
| 2 | Channel B fine pitch | 8 bits | Tone period for channel B (low byte) |
| 3 | Channel B coarse pitch | 4 bits | Tone period for channel B (high nibble) |
| 4 | Channel C fine pitch | 8 bits | Tone period for channel C (low byte) |
| 5 | Channel C coarse pitch | 4 bits | Tone period for channel C (high nibble) |
| 6 | Noise pitch | 5 bits | Noise period (0 = quiet, 1 = highest pitch) |
| 7 | Mixer | 8 bits | Enables per-channel tone/noise: bit 0 = tone A, 1 = tone B, 2 = tone C, 3 = noise A, 4 = noise B, 5 = noise C (0 = enable, 1 = disable) |
| 8 | Channel A volume | 5 bits | Bits 0–3 = volume (0–15), bit 4 = envelope enable (1 = use envelope instead of fixed volume) |
| 9 | Channel B volume | 5 bits | Same as register 8 |
| 10 | Channel C volume | 5 bits | Same as register 8 |
| 11 | Envelope fine duration | 8 bits | Envelope period (low byte) |
| 12 | Envelope coarse duration | 8 bits | Envelope period (high byte) |
| 13 | Envelope shape | 4 bits | Envelope waveform shape (see below) |
| 14 | I/O port A | 8 bits | General-purpose I/O (connected to RS-232/MIDI and keypad on 128K) |
| 15 | I/O port B | 8 bits | Not connected on AY-3-8912 |

### Envelope Shapes (Register 13, bits 0–3)

| Bit pattern | Description |
|---|---|
| `0 0 0 0` | \___ | Single decay then off |
| `0 1 0 0` | \___ | Same |
| `1 0 0 0` | /___ | Single attack then off |
| `1 1 0 0` | /___ | Same |
| `0 0 1 0` | \\\\\ | Continuous decay |
| `0 1 1 0` | \\\\\ | Same |
| `1 0 1 0` | ///// | Continuous attack |
| `1 1 1 0` | ///// | Same |
| `0 0 0 1` | \_/\\_ | Decay-attach-decay-attack |
| `0 1 0 1` | \_/\\_ | Same |
| `1 0 0 1` | /\\_/\ | Attack-decay-attack-decay |
| `1 1 0 1` | /\\_/\ | Same |
| `0 0 1 1` | \_\_\_ | Continuous decay with sustain (same as 0x0A) |
| `0 1 1 1` | \_\_\_ | Same |
| `1 0 1 1` | /_/_/ | Continuous attack with sustain |
| `1 1 1 1` | /_/_/ | Same |

Bit 0 of register 13 controls whether the envelope resets on each key-on (0 = hold, 1 = continue). Bit 2 selects hold (0) vs alternate (1). Bits 1 and 3 control attack/decay direction.

## Audio Distortion on +2A/+3

The +2A and +3 (Z70830 board) have a design flaw in the audio output circuitry that causes serious distortion to AY-3-8912 music. This was fixed in the **+2B** and **+3B** revisions (Z70833 and Z70835 boards respectively), which use a redesigned audio circuit.

The +2B board Z70833 further has ISSUE 4 variants ©1990 that use a different FM modulator circuit for audio output.

Emulators that target cycle-accurate sound should model the +2A/+3 audio distortion or implement the cleaner +2B/+3B output as a configurable option.
