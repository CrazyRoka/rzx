# AY-3-8912 Sound Chip (128K)

The General Instrument AY-3-8912 programmable sound generator (PSG) is present in all post-48K models (128K, +2, +2A, +2B, +3). It is controlled by two I/O ports:

| Port | Direction | Function |
|---|---|---|
| `0xFFFD` | OUT | Select register (0–14) |
| `0xFFFD` | IN | Read value of the selected register |
| `0xBFFD` | OUT | Write value to the selected register |

Usage: select a register by writing to `0xFFFD`, then read/write the register value via `0xBFFD` (read) or `0xFFFD` (write).

## Pinout (28-pin DIP)

```
            +------+
  SOUND C  1 |      | 28  D0
  PORT     2 |      | 27  D1
  Vcc      3 |      | 26  D2
  SOUND B  4 |      | 25  D3
  SOUND A  5 |      | 24  D4
  GND      6 |      | 23  D5
  PORT     7 |      | 22  D6
  PORT     8 |      | 21  D7
  PORT     9 |      | 20  BC1
  PORT    10 |      | 19  BC2
  PORT    11 |      | 18  BDIR
  PORT    12 |      | 17  A8
  PORT    13 |      | 16  RESET
  CLOCK   14 |      | 15  CLOCK
            +------+
```

| Pin(s) | Name | Function |
|---|---|---|
| 1 | SOUND C | Audio output channel C |
| 2, 7–12 | PORT | I/O port connections to the system |
| 3 | Vcc | +5 V power supply |
| 4 | SOUND B | Audio output channel B |
| 5 | SOUND A | Audio output channel A |
| 6 | GND | Ground |
| 13 | RESET | Reset (active low) |
| 14 | CLOCK | Clock input (system clock divided down) |
| 15 | CLOCK | Clock output |
| 16 | RESET | Reset |
| 17 | A8 | Address line 8 (register select) |
| 18 | BDIR | Bus direction control |
| 19 | BC2 | Bus control 2 |
| 20 | BC1 | Bus control 1 |
| 21–28 | D0–D7 | Data bus |

In the Spectrum, the three audio outputs (SOUND A, B, C) are typically tied together.

### Clock Note

In the **Timex TS2068**, the AY-3-8912 runs at **1.76475 MHz** rather than the standard Spectrum clock.

## Audio Distortion on +2A/+3

The +2A and +3 (Z70830 board) have a design flaw in the audio output circuitry that causes serious distortion to AY-3-8912 music. This was fixed in the **+2B** and **+3B** revisions (Z70833 and Z70835 boards respectively), which use a redesigned audio circuit.

The +2B board Z70833 further has ISSUE 4 variants ©1990 that use a different FM modulator circuit for audio output.

Emulators that target cycle-accurate sound should model the +2A/+3 audio distortion or implement the cleaner +2B/+3B output as a configurable option.
