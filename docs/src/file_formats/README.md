# File Formats

This chapter documents the file formats used to store Spectrum data for emulation: snapshot images, tape recordings, disk images, screen dumps, and Microdrive cartridges.

## Subsections

- [SNA Snapshot Format](./sna_format.md) — 48K and 128K memory snapshots
- [SCR Screen Format](./scr_format.md) — screen pixel + attribute dumps
- [MDR Microdrive Format](./mdr_format.md) — ZX Microdrive cartridge images
